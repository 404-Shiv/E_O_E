# EOE – Financial News Verification & Investment Assistant

EOE (End Of Evaluation) is a simple yet practical machine learning project that helps users **check whether financial news is TRUE or FAKE** and also assists in **basic investment decisions** like stock performance and mutual fund details.

The idea behind this project is to reduce the impact of misleading financial news and help users make more informed decisions using data and machine learning.

---

## What This Project Does

This system allows users to:

- **Manually type a financial news headline** and check:
  - Whether it is TRUE or FAKE
  - How confident the model is (in percentage)

- **Analyze live financial news**
  - Fetches real-time news from trusted Indian business sources
  - Automatically classifies them into TRUE and FALSE

- **Check stock performance**
  - Tells whether a stock is GOOD, NEUTRAL, or UNDERPERFORMING
  - Uses historical market data

- **View mutual fund information**
  - Fetches latest NAV data from AMFI (official source)

All of this is done using a **menu-driven program**, so you don’t need to run multiple commands again and again.

---

## Why This Project Is Useful

- Fake or misleading financial news can cause wrong investment decisions
- This project shows how **machine learning can help verify information**
- It combines **news verification + finance**, making it very practical
- No API keys are required — safe and easy to run
